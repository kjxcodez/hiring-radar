# hiring-radar

A local-first Python CLI that discovers companies actively hiring by querying
public ATS APIs and job boards (Greenhouse, Lever, Ashby, WorkMap, etc.),
then structures the results — company name, open roles, tech stack signals,
and contact hints — into clean JSON/CSV output ready for manual cold outreach.
No SaaS, no cloud infra: everything runs on your machine.

> **Status: early WIP** — scaffolding only; no commands are implemented yet.
